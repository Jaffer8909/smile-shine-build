

# Vivek Dental — Premium Specialist Website

## Design Direction: "Clinical Zen"
A high-end medical minimalism aesthetic inspired by Aēsop and Apple Health. Clean, calming, and trust-building — no generic "medical blue" or stock smile photos.

**Typography:** Instrument Serif (headlines) + Instrument Sans (body)  
**Palette:** Off-white mint background, deep forest teal primary (#2D5A56), soft mint accents  
**Feel:** Matte ceramic surfaces, generous whitespace, subtle scroll animations

---

## Pages & Structure

### 1. Homepage (The Trust Path)
- **Navigation:** Transparent → solid on scroll. Links: Services, About Dr. Vivek, Contact. CTA: "Request a Visit" button
- **Hero:** Left-aligned headline like *"Specialist Dental Care. Gentle Precision."* with sub-text about Dr. Vivek's endodontics expertise. Right side uses existing clinic/doctor imagery
- **Trust Bar:** "4.9★ Google Rating · MDS Endodontics · 15+ Years · 10,000+ Patients"
- **Services Grid:** 3-column cards — Root Canal, Smile Design, Tooth Whitening, Implants, Orthodontics, Oral Surgery — with clean line-art icons
- **The Doctor Section:** Split layout — Dr. Vivek's photo (from existing site) + credentials, philosophy, and specializations
- **Booking Section:** Centered card with simple form (Name, Phone, Preferred Date/Time, Treatment) + WhatsApp and Call buttons
- **Sticky Mobile CTA:** Fixed bottom bar with "Call Now" and "WhatsApp" buttons

### 2. Individual Service Pages (6 pages)
Each treatment gets its own page for SEO:
- Root Canal Treatment
- Smile Designing / Cosmetic Dentistry
- Tooth Whitening
- Dental Implants
- Orthodontics / Braces
- Oral Surgery

**Each page includes:** Treatment overview, what to expect, recovery info, FAQ accordion, and a "Request a Visit" CTA

## Conversion Features
- **Triple booking options** on every page: Form, WhatsApp button, Click-to-call
- **Sticky mobile CTA bar** (call + WhatsApp) visible on all pages
- **SEO-optimized** service pages targeting "root canal cost in Hyderabad," "best dentist in Hyderabad" etc.
- **Scroll animations:** Subtle fade-in with spring physics for a premium feel
- **Mobile-first:** Designed for the 70%+ of patients browsing on phone

## Technical Notes
- Built with React + Tailwind CSS + shadcn/ui components
- Google Fonts: Instrument Serif + Instrument Sans
- Fully responsive, fast-loading single-page app with client-side routing
- Form data displayed as a success toast (no backend needed initially — can add Supabase later for storing leads)
